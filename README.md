# CRA Customized

Just some adjustments to the default CRA setup :)

## Tweaks

- Heroku setup
- HMR support
- Storybook
- Standard.js
- Alias + VS Code intellisense support (check `jsconfig.json`)
- Webpack Bundle Analyzer

- yarn build
- yarn analyze
